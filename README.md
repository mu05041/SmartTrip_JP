> "旅行に行きたいけど、旅行の日程を組むのが大変で面倒くさい...どうしよう？"
>

### そんな計画が苦手な人のための旅行プランナー、モヨ
- 旅行の日程もモヨにお任せください！ 
- 旅行に行く仲間同士で日程を編集できます！ 
- 旅行が終わったら、フォトアルバムで思い出を共有しましょう！

&nbsp;
